Чтобы автоматически выводить список товаров из каталога Tilda в Zero Block без ручного прописывания ID для каждого товара,
используется связка одного элемента-шаблона в Zero Block и специального скрипта (кода).Скрипт сам посчитает,
сколько товаров в каталоге, размножит ваш дизайн из Zero Block и подставит туда правильные названия, картинки и тексты.
Вот пошаговый план настройки:Шаг 1. Создайте один шаблонный товар в Zero BlockВам нужно нарисовать дизайн всего одной карточки товара.
Добавьте Zero Block. Создайте в нем одну карточку: элемент Image (для фото), Text (для названия), Text (для описания) и Button (для кнопки «Купить»).
Задайте элементам внутри Zero Block универсальные CSS-классы (нажмите правой кнопкой мыши 
на элемент -> Add CSS Class Name):Для картинки: js-custom-imgДля названия: js-custom-titleДля описания/текста: js-custom-textДля цены (если нужна):
js-custom-priceДля кнопки «Купить»: js-custom-btnВыйдите из Zero Block. В левом верхнем углу блока скопируйте его ID (он выглядит как #rec123456789).

В его контенте выберите нужный раздел товаров (сколько товаров выведется в этом блоке, столько и появится в вашем Zero Block).Скройте этот блок,
